Введение в Frontend - разработку

Что такое Frontend и зачем он нужен ?

Frontend -это чвсть веб разработки, которая отвечает за визуальную и интерактивную часть сайтов и веб приложений. Всё что вы видите в браузере- кнопки, формы, анимации, меню ,шрифты, изображения- создается с помощью HTML,CSS JavaScript.Именно frontend делает сайт живым ,удобным для пользователей и понятным для взаимодействия.
Frontend тесно связвн с UX/UI-дизайном и играет ключевую роль в восприятии продукта пользователем.
Основные технологии которые использует frontend-разработчик:
  • HTML - структура и страницы
  • CSS - стили и оформление
  • LavaScript - логика и поведение элементов

Что нужно знать начинающему фронтендеру?

Путь в frontend начинается с изучения базовых технологий На начальном этапе важно научиться верстать простые, страницы подключать стили, писать базовые скрипты .Кроме трёх основных языков стоит обратить внимание на такие инструменты как :
  1. Git - система контроля версий
  2. npm - менеджер пакетов
  3. React,Bootstrap - популярные библиотеки и фреймворки
Также важно понимать такие концепции, как адаптивная верстка семантический HTML и доступность интерфейсов(a11y).

Краткое описание технологий

Основные языки
HTML(HyperText Markur Language) - язык разметки, с помощью которого создается структкра веб- страницы: заголовки, параграфы,, списки изображения и другие элементы.
CSS(Cascading Style Sheets) - язык стилей, позволяющий задавать цвета, шрифты,, фон отступы и другие визуальные свойства элементов.
JavaScript - язык программирования, который позволяет оживить страницу: создавать интерактивные элементы ,анимации и обработку событий.

Инструменты разработки

Git - система, контроля версий, позволяющая отслеживать изменения в коде, работать в команде и управлять историями изменений.
