Введение в Frontend-разработку

Что такое Frontend и зачем он нужен?

Frontend- это часть веб разработки, которая отвечает за визуальную и интерактивную часть сайтов и веб приложений. Все что вы видите в браузере ,кнопки, формы ,анимации, меню ,шрифты, изображения -создается с помощью HTML, CSS ,JavaScript.Именно Frontend делает сайт живым удобным для пользователей и понятным для взаимодействия

Frontend тесно связан с UX/UI-дизайном и играет ключевую роль в восприятии продукта пользователем. Основные технологии которые использует frontend-разработчик:

  • HTML-структура страницы
  • CSS-стили и оформление
  • JavaScript-логика и поведение элементов

Что нужно знать начинаещему фронтендеру

Путь во Frontend начинается с базовых технологий. На начальном этапе важно научиться верстать простые страницы, подключать стили,. писать базовые скрипты Кроме трех основных языков ,стоит обратить внимание на такие инструменты как:

  2. Git-система контроля версий
  3. npm-менеджер пакетов
  4. React Bootstrap-популярные библитеки и фреймворки
Также важно понимать такие конйепции ,как адаптивная верстка, семантический HTML и доступность интерфейсов (a11y).

Краткое описание технологий

Основные языки

HTML (HyperText Markur Languuage) -язык разметки ,с помощью которого создается -структура веб- страницы,заголовки, параграфы ,списки, изображения и другие элементы

CSS(Cascading Style Sheets) -язык стилей ,позволяющий задать цвета, фрифты отступы и другие визуальные свойства элнментов.

JavaScript-язык пограммирования, который позволяет оживить страницу, создавать интнрактивные элементы, анимации и обработку событий.

Инструменты разработки

Git-система контроля версий, позволяющая отслеживать изменения в коде, работать в команде и управлять историями изменений.

npm-менеджер пакетов ,используемый в JavaScript проэктах, для установки библиотек и управления зависимостями.

фреймворки и библитеки

React-библиотека для создания пользовательских интерфейсов с использованием компонентов и виртуального DOM

Bootstrap-CSS фреймворк, с готовыми компонентами и сеткой ускоряющий, разработку адаптивных интерфейсов.
